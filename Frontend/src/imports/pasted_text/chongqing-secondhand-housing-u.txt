请为一个本科课程学年设计项目生成一套完整的 Web 后台系统 UI 设计稿，系统名称为：

《重庆市二手房源价格数据分析与智能可视化系统》

项目归属：
西南大学 · 商贸学院 · 学年设计Ⅱ

系统定位：
这是一个面向重庆市二手房挂牌价数据的数据分析与智能可视化后台系统，不是普通管理系统。系统需要体现“西南大学蓝白科研风 + 重庆山城城市元素 + 房价数据分析场景”，整体风格正式、简约、科研感强，适合毕业答辩、课程设计展示和顶会 Demo 式系统演示。

一、整体视觉风格要求

1. 主风格：

* 西大蓝白科研风
* 白底、深蓝标题、浅灰背景、蓝色交互
* 科研、正式、简约、清爽、可信
* 不要赛博朋克，不要过度炫酷，不要黑色大屏风
* 像“高校科研项目 Demo 系统 + 现代数据分析后台”

2. 品牌元素：

* 左上角使用“西南大学”文字与圆形校徽占位符，不要生成复杂错误校徽，使用圆形线性图标占位即可
* 可在登录页、系统顶部、报告预览页加入“1906”浅色水印
* 可使用小字文案：“含弘光大 · 继往开来”或“特立西南 · 学行天下”
* 系统归属文案：“西南大学 · 商贸学院 · 学年设计Ⅱ”

3. 重庆城市元素：

* 使用浅蓝线稿形式加入重庆城市天际线、桥梁、两江、山城层叠地形、重庆区县地图
* 不要用大面积实景照片抢视觉
* 重庆元素主要用于登录页背景、顶部装饰、Dashboard 地图、报告封面

4. 推荐配色：

* 主色深蓝：#163A70
* 品牌蓝：#1F4E8C
* 辅助蓝：#4F7DBD
* 页面背景：#F7F9FC
* 卡片背景：#FFFFFF
* 边框灰：#E5EAF2
* 正文深灰：#1F2937
* 次级文字：#6B7280
* 成功绿：#16A34A
* 警示橙：#F59E0B
* 异常红：#DC2626
* 重庆暖橙点缀：#E67E22

5. 字体与布局：

* 中文字体使用思源黑体、苹方或系统无衬线字体
* 标题加粗，正文清晰
* 统一 8px 栅格系统
* 卡片圆角 12px
* 页面间距充足
* 细边框、轻阴影，不要厚重阴影

设计一个 Web 后台管理系统，统一布局如下：

* 主内容区使用卡片式布局

页面 6：智能问答与报告页

页面目标：
体现 DeepSeek Agent 通过工具调用读取数据，生成可追溯回答和报告。

布局：
三栏结构：
左侧：会话历史
中间：智能问答聊天区
右侧：工具调用 Trace + 报告预览

会话历史：

* 重庆各区均价分析
* 渝北区趋势分析
* 刚需房源价格分析
* 异常房源检测
* 生成月度报告

聊天区：
用户问题示例：
“请分析渝北区近一年的二手房挂牌均价走势，并与主城区其他区县对比。”

AI 回答示例：
“根据系统数据，渝北区近一年挂牌均价整体呈小幅震荡趋势，当前均价为 14,489 元/㎡，低于渝中区和江北区，高于巴南区和北碚区。”

工具调用 Trace：

* query_market_stats：查询区县均价
* get_chart_series：获取趋势图数据
* get_model_result：读取模型分析结果
* generate_report：生成报告

报告预览：
标题：
渝北区二手房挂牌价趋势分析报告

报告内容：

* 核心结论
* 数据范围
* 价格趋势图
* 区县对比图
* 关键指标表
* 数据来源与证据
* 下载 PDF 按钮
* 导出 Word 按钮

强调：
AI 回答必须有证据来源区域，不要像普通聊天机器人。
